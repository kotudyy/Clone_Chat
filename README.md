Firebase에서 제공하는 회원가입, 로그인 함수를 이용해서 Firebase Database에 사용자의 Email 정보를 저장하고,
채팅을 이용하면 사용자의 FCM Token을 저장한다.
사용자의 정보는 Firebase Database의 컬렉션 'profile' 내 사용자의 Email을 이름으로 하는 파일에 저장된다.
